# HealthSense-ML

Đây là kho chứa mã nguồn dành riêng cho việc phân tích dữ liệu (Data Science) và huấn luyện mô hình Machine Learning (Random Forest) cho dự án HealthSense.

## Cấu trúc thư mục
- `data/raw/`: Chứa các file CSV gốc thu thập từ ESP32.
- `data/processed/`: Chứa các file CSV sau khi đã làm sạch và trích xuất đặc trưng.
- `notebooks/`: Chứa các file Jupyter Notebook để chạy thử nghiệm và phân tích.
- `models/`: Chứa các mô hình đã được huấn luyện.

## Cài đặt môi trường
1. Cài đặt Python 3.9+
2. Chạy lệnh: `pip install -r requirements.txt`
3. Mở Jupyter Notebook bằng lệnh: `jupyter notebook`
